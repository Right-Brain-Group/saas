## S.C.I.E.N.C.E.

[S]{.font-bold}trategic: Implementing well-thought-out strategies for web design to ensure goals are met.

[C]{.font-bold}reative: Utilizing unique and imaginative design elements to stand out.

[I]{.font-bold}nnovative: Leveraging the latest technologies and trends in web development.

[E]{.font-bold}ngaging: Crafting designs that captivate and hold users' attention.

[N]{.font-bold}avigable: Ensuring easy and intuitive navigation for an optimal user experience.

## Features

- Powered by [Nuxt 3](https://nuxt.com)
- Built with [Nuxt UI](https://ui.nuxt.com) and [Nuxt UI Pro](https://ui.nuxt.com/pro)
- Write content with [MDC syntax](https://content.nuxt.com/usage/markdown) thanks to [Nuxt Content](https://content.nuxt.com)
- Beautiful Typography styles
- Full-Text Search out of the box
- Dark mode support
- And more...
